# marktschopp.github.io ✦

Source for **[marktschopp.github.io](https://marktschopp.github.io)** — the professional site of
**Mark Tschopp, PhD**: scientist, engineer, educator, and leader at the DEVCOM Army Research
Laboratory, working at the intersection of materials science, solid mechanics, machine learning &
AI, and the science of science.

## The site

A five-page static site — no frameworks, no trackers, no build step in the browser:

| Page | What's there |
|---|---|
| [Home](https://marktschopp.github.io) | Bio, research areas, selected publications, recognition |
| [Career](https://marktschopp.github.io/career.html) | Two decades across industry, academia, and government |
| [Publications](https://marktschopp.github.io/publications.html) | The full peer-reviewed record |
| [Presentations](https://marktschopp.github.io/presentations.html) | Invited seminars and conference talks |
| [Teaching](https://marktschopp.github.io/teaching.html) | Courses, tutorials, and outreach |

A current [CV (PDF)](https://marktschopp.github.io/Tschopp-CV.pdf) is linked from the home page.

## How it's built

Plain HTML and CSS with a deep-space, *Star Wars*–inspired design language. The publication,
presentation, and teaching lists are generated from a maintained LaTeX CV, so the site stays in
sync with the CV of record.
